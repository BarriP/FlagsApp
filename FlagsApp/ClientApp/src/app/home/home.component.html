¿Como funciona?

  • Fases de la prueba: Primero se hará un test inicial para comprobar los conocimientos, despues pasaremos a las 8 rondas de entrenamiento y finalmente una prueba final
  • Tests: En ambos test se mostraran las 50 banderas de los estados de EEUU y habrá un desplegable para elegir una unica respuesta. Se deben rellenar todas las preguntas para continuar. A continuación se mostrarán los resultados y pasaremos a la fase de entrenamiento
  • Entrenamiento: El entrenamiento se divide en 8 rondas con 8 preguntas cada una. Se mostrará una bandera y escogerás la opción que consideres correcta de las mostradas. Inmediatamente se mostrará el resultado. Al final de cada ronda se mostrarán de nuevo las banderas de cada ronda para repasar
  • Revisiones: Tomate el tiempo que necesites para analizar tus fallos (y tus aciertos). El objetivo es que aprendas a utilizando la aplicación 😊. Cuando estes listo, continua con el test
  • Tiempo: No hay limite de tiempo para contestar las preguntas ni en las revisiones, ni en los tests. El tiempo limite es el consideres oportuno
  • Desconexion: Si te desconectas en mitad de la prueba o no terminas todas las fases, los datos no se guardarán y será necesario repetir la prueba

Los datos introducidos se tratarán de manera anonima para fines estadísticos
